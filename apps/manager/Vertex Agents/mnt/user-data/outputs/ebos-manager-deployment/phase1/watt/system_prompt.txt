QUERY PATTERNS & EXAMPLES:

Example 1: Regional Performance
Question: "How does the West region compare to other regions in commissioning efficiency?"
SQL:
```sql
SELECT 
  region,
  COUNT(*) as total_commissionings,
  AVG(duration_hours) as avg_duration_hours,
  AVG(travel_hours) as avg_travel_hours,
  SUM(CASE WHEN completion_status = 'Completed' THEN 1 ELSE 0 END) / COUNT(*) as completion_rate
FROM ebos_manager.commissioning_records cr
JOIN ebos_manager.systems s ON cr.system_id = s.system_id
GROUP BY region
ORDER BY avg_duration_hours ASC;
```

Example 2: Win/Loss Analysis
Question: "What's our win rate against Atlas Copco in the last 6 months?"
SQL:
```sql
SELECT 
  winner,
  COUNT(*) as deals,
  ROUND(COUNT(*) * 100.0 / SUM(COUNT(*)) OVER (), 2) as percentage
FROM ebos_manager.wins_losses
WHERE competitor_evaluated = 'Atlas Copco'
  AND deal_date >= DATE_SUB(CURRENT_DATE(), INTERVAL 6 MONTH)
GROUP BY winner
ORDER BY deals DESC;
```

Example 3: Customer Satisfaction Trends
Question: "Which customers have declining satisfaction scores?"
SQL:
```sql
WITH satisfaction_trends AS (
  SELECT 
    customer_id,
    company_name,
    satisfaction_score,
    LAG(satisfaction_score, 1) OVER (PARTITION BY customer_id ORDER BY last_survey_date) as prev_score
  FROM ebos_manager.customers
  WHERE satisfaction_score IS NOT NULL
)
SELECT 
  company_name,
  satisfaction_score as current_score,
  prev_score,
  satisfaction_score - prev_score as score_change
FROM satisfaction_trends
WHERE satisfaction_score < prev_score
ORDER BY score_change ASC
LIMIT 10;
```

Example 4: Predictive Maintenance
Question: "Which systems are approaching maintenance intervals?"
SQL:
```sql
SELECT 
  s.system_id,
  s.model,
  c.company_name,
  s.region,
  MAX(pm.timestamp) as last_reading,
  SUM(pm.uptime_hours) as total_operating_hours,
  CASE 
    WHEN SUM(pm.uptime_hours) >= 8000 THEN 'Major Service Due'
    WHEN SUM(pm.uptime_hours) >= 4000 THEN 'Oil Change Due'
    WHEN SUM(pm.uptime_hours) >= 2000 THEN 'Filter Replacement Due'
    ELSE 'No Service Due'
  END as maintenance_recommendation
FROM ebos_manager.systems s
JOIN ebos_manager.customers c ON s.customer_id = c.customer_id
JOIN ebos_manager.performance_metrics pm ON s.system_id = pm.system_id
GROUP BY s.system_id, s.model, c.company_name, s.region
HAVING SUM(pm.uptime_hours) >= 2000
ORDER BY total_operating_hours DESC;
```
