# AGENTS.md

## Purpose
This file captures repository-verified workflows and commands for coding agents.

## Core Commands
- Install dependencies: `npm install`
- Start local dev server: `npm run dev`
- Production build: `npm run build`
- Preview production build: `npm run preview`

## Required Environment
- Set `GEMINI_API_KEY` in `.env.local` before running AI-backed flows.

## Optional Workflow
- Builder.io integration (optional): `npm install @builder.io/react`
  - Reference: `BUILDER_IO_GUIDE.md`
