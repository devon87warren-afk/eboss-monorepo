# BrainGrid Rules

Apply these defaults whenever BrainGrid MCP, BrainGrid CLI, or BrainGrid requirement and task workflows are relevant in this repository.

## Defaults

- Use intent-first BrainGrid workflows. Optimize for what the user is trying to accomplish, not the literal tool name they mention.
- Discover project context first when it is missing. Reuse `.braingrid/project.json` when available.
- Follow the routing and completion rules in `BRAINGRID_MCP.md`.
- After analysis is complete, continue the BrainGrid workflow to the next meaningful action unless the user explicitly asks to stop at analysis.
